# Web Dev Teaching Repo (HTML • CSS • JavaScript)

Lightweight starter for teaching HTML, CSS, and JavaScript — no build tools required.

## Structure
```
.
├── index.html
├── styles.css
├── script.js
├── lessons/
│   ├── 01-html-basics/
│   ├── 02-css-basics/
│   ├── 03-js-basics/
│   ├── 04-dom-and-events/
│   └── 05-mini-projects/
└── .editorconfig
```

## Run
Use VS Code Live Server, or:
```bash
python -m http.server 5500
```
Open http://localhost:5500
